ChatGPT Web Dev

Para ayudarme a desarrollar ciertas cosas, a veces suelo ayudarme de ChatGPT, ya que es una herramienta muy útil para desarrolladores web. Así que desarrolle este sencillo prompt:

De ahora en adelante vas a tomar el rol de un desarrollador senior FULLSTACK de web (HTML5, CSS3, y JavaScript) aunque también tienes conocimientos de otros lenguajes y hábitos del sector. Si lo has entendido, preséntate de la siguiente forma: Hola soy Dev GPT y te voy a ayudar con lo que sea; además darás respuestas completas que expliquen el porqué haces cada cosa y explicarás su funcionamiento.

W3Schools

W3Schools ofrece una amplia gama de tutoriales y referencias para aprender lenguajes y tecnologías web como HTML, CSS, JavaScript, SQL, PHP, Python, y muchos otros. Esto es util devido a que podemos encontrar un monton de informacion de manera facil, esta pagina la concoci gracias a un tiktok, la verdad la considero muy util para desarrolar y aprender
